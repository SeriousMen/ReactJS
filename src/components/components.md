### 함수 선언방식

```
1. 함수 선언식
EX) function foo() {return 5;}

2. 함수 표현식
EX) var foo = function () {return 5;};

함수 선언식은 코드가 실행되기 전에 로드되지만(호이스팅), 함수 표현식은 인터프리터가 해당 코드 줄에 도달할 때만 로드된다.
EX)
var foo = function () {return 5;}
이전에 foo(); 를 작성하면 에러가 발생한다.

함수 선언문은 코드 블록 내에 위치하면 블록 밖에서 해당 함수에 접근하지 못한다. 

함수 표현식을 이용해서 코드 블록 밖에서 선언한 함수에 함수를 할당하면 외부에서도 접근이 가능하다. 

ES6에서 추가된 화살표 함수 방식을 정의할 수 있다. 
EX) var foo = (x,y) => (x+y)

*호이스팅 : 함수 안에 있는 선언들을 모두 끌어올려서 해당 함수 유효 범위(scope)의 최상단에 위치하는 것을 말한다.

*js 스코프
1. var : 선언된 함수 범위내, 함수 밖에서 선언되면 전역, 재선언/ 값 변경 가능, 호이스팅, 호이스팅 시에 undefined으로 초기화 됨
2. let : block({})내 범위, 같은 범위 내에서(블록) 재선언 안됨, 값 변경은 가능, 호이스팅 호이스팅 시에 초기화 안됨
3. const : block({})내 범위, 같은 범위 내에서)블록 재선언 안됨, 값 변경도 안됨 but 개체 속성 값은 변경가능, 선언과 동시에 초기화 해야함, 호이스팅, 호이스팅 시에 초기화 안됨 
```
